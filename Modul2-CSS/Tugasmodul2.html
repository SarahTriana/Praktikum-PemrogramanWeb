CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan layout halaman web. CSS memungkinkan kita mengubah warna, ukuran, posisi, dan gaya elemen HTML agar halaman web menjadi menarik dan mudah dibaca. CSS digunakan untuk mengatur tampilan elemen HTML di berbagai media seperti yang tertampil pada layar komputer atau perangkat lainnya. CSS memungkinkan pengembang web untuk mengontrol tata letak banyak halaman sekaligus sehingga menghemat waktu dan usaha dalam desain web.
